# Installation Guide

UniAIMS is designed to run in a containerized environment, but can also be installed locally.

## Prerequisites

*   Node.js v18+
*   Python 3.9+ (for backend inference)
*   CUDA-compatible GPU (recommended)

## Quick Install

\`\`\`bash
# Clone the repository
git clone https://github.com/uniaims/platform.git

# Install dependencies
npm install

# Start the development server
npm run dev
\`\`\`
